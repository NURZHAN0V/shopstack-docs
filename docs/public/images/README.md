# Скриншоты документации

Положите PNG/WebP в подпапки `users/`, `storefront/`, `api/`.

После добавления файла в markdown установите `status="ready"` у компонента `<DocScreenshot>`.

Список ожидаемых файлов — в разделе [Правила портала docs](/v1.0/ru/guides/best-practices/documentation-portal) (скриншоты).
