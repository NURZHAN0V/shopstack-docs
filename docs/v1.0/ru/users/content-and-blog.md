# Контент и блог

Модуль **Контент** — блочный редактор статей, страницы, медиатека, виджеты.

## Редактор статей

Блоки: текст, заголовки, изображения, таблицы, LaTeX, кнопки, аккордеон.

**Медиа:** изображения, встраивание VK Video, RUTUBE, YouTube (ссылка → iframe).

**Виджеты:** Google / Яндекс Карта по ссылке.

## AI-помощник

При настроенном `OPENROUTER_API_KEY`:

- генерация целой статьи по запросу;
- доработка фрагмента в выбранном блоке.
