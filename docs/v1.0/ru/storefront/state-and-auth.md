# Состояние и авторизация

## Pinia

Состояние между страницами: корзина, пользователь, настройки сайта. Имена сторов с префиксом `use`: `useUser`, `useCart`.

## JWT на клиенте

- Токен не хранить в открытом виде в localStorage без необходимости; предпочтительнее httpOnly cookie через server route
- При **401** — logout и редирект на login
- Скрытие кнопок в UI **не заменяет** проверку прав на Go API

## Cookies

По `goshop_client/AGENTS.md`: настройки и несекретные prefs — cookies; пароли и сырые токены — нет.

## Provide/inject

Только для theme/i18n на уровне layout, не для бизнес-данных каталога.
