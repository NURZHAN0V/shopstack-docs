# Дорожная карта e-commerce

ShopStack наращивается **поэтапно**: после каждого этапа — рабочий срез, а не заготовки «на потом».

## Этапы

| № | Этап | Содержание |
|---|------|------------|
| 1 | Витрина и каталог | Список, карточка, фильтры, поиск |
| 2 | Бэкенд и заказы | API, корзина, оформление, админ заказов |
| 3 | Поиск и фильтры | FTS, атрибуты, сравнение |
| 4 | UX и тема | Skeleton, тема панели, брендинг магазина |
| 5 | Авторизация | Регистрация, JWT, личный кабинет, 2FA админов |
| 6 | Оплата и доставка | ЮKassa, T-Bank, СДЭК |
| 7 | Контент | Блог, блоки, AI в редакторе |
| 8 | SEO витрины | Опционально (сейчас CSR + noindex по политике витрины) |
| 9 | Расширения | CRM, мультивалюта — post-MVP |

## Статус ShopStack

| Этап | Статус |
|------|--------|
| API + каталог | Готово |
| Заказы, оплата, СДЭК | Готово |
| Контент + AI | Готово |
| Nuxt витрина (CSR) | Готово |
| SEO витрины | Отключён намеренно |
