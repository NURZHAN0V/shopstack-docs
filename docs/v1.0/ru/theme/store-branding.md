# Брендинг магазина

**Тема** в модуле контента — настройки **витрины**, а не светлая/тёмная UI панели.

## Поля

- Логотип (медиатека)
- Название сайта

Используются в шапке Nuxt-витрины и в превью контента (`VITE_STOREFRONT_URL` в панели).
