# Матрица совместимости

Версии компонентов ShopStack на одну дату могут различаться — сверяйтесь при деплое и интеграциях.

| Компонент | Где указана версия | Текущий пример |
|-----------|-------------------|----------------|
| **API (контракт)** | `swagger.yaml`, `main.go` @version | **1.0** |
| **Панель (npm)** | `goshop/package.json` | **0.1.0** |
| **Docker prod** | `deploy/.env.example` → `IMAGE_TAG` | **1.0.0** |
| **Витрина** | `goshop_client/package.json` | **1.0.0** |
| **Портал docs** | `shopstack-docs/package.json` | **1.0.0** |
| **Снимок docs (URL)** | `docs/v1.0/`, переключатель в шапке | **v1.0** (API 1.0) |

## Политика изменений API

| Тип | Ожидание |
|-----|----------|
| **Patch** | Обратная совместимость, исправления |
| **Minor** | Новые эндпоинты, опциональные поля |
| **Major** | Breaking changes (отдельное объявление) |

## Проверка на стенде

1. `IMAGE_TAG` API/nginx совпадает с релизными notes
2. Версия API в `swagger.yaml` / документации портала
3. Витрина: `NUXT_PUBLIC_API_URL` указывает на тот же major API
