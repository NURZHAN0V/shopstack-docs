# Storefront conventions

Source of truth: `goshop_client/AGENTS.md`.

| Topic | Typical Vue SPA | ShopStack storefront |
|-------|-----------------|----------------------|
| Framework | Vue 3 + Vite | **Nuxt 4** |
| Styling | Custom CSS | **Tailwind** (`warm`, `teal`, `ink`) |
| Icons | Inline SVG | **Remix Icon only** |
| Rendering | SSR/SSG | **CSR only** |
| SEO | Indexed | **noindex** |
| PWA | Optional | Not enabled yet |

See [Styling and UI](./styling-and-ui.md).
